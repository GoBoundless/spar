Spar

Single Page App Rocketship

Ignition

Spar is an opinionated framework that aims to ease single-page web-app development by addressing the major challenges:

  • Asset organization
  • Compilation pipeline
  • Building & deploying

Spar is designed for apps primarily built with HTML, CSS & Javascript, and thus is platform agnostic.

We’ve got some examples to get you going.

Launch Codes

  gem install spar
  spar new YOUR_APP
  cd YOUR_APP
  spar server

Lift-off!

  spar deploy production
